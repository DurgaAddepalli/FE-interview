### 前端组件库本地开发调试方案优化，模块化调试技巧？  

使用npm link

```
npm link   //组件文件夹：全局引用， 会在 node_modules中生成 npm-link-module的快捷方式，被链接到全局
npm link npm-link-module   //项目文件夹：后面是全局引用文件夹
```
